# Week 4 - Question 1: Marquee

## Setup Instructions

1. **Navigate to this directory:**
   ```bash
   cd week4/q1
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm start
   ```

4. **Open your browser** and go to `http://localhost:3000`
